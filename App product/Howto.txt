How to run this web application
1. Install the following libraries : flask, chromed, json, tqdm
2. Open and Run app_chroma.py in Visual Studio Code
3. Wait for DB to be loaded
4. In the terminal there should be "* Running on http://127.0.0.1:5001"
5. Copy and paste http://127.0.0.1:5001 onto web browser to run a website
