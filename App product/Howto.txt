How to run this web application
1. Install the following libraries : flask, chromadb, json, tqdm
2. Open and Run app_chroma.py in Visual Studio Code
3. Wait for DB to be loaded
4. In the terminal there should be "* Running on http://127.0.0.1:5001"
5. Copy and paste http://127.0.0.1:5001 onto web browser to run a website

This database is slightly outdated in terms of features and content.
Currently there is not a database of this size for the features group 3 added.
You could make it yourself, but if you are testing the app_chroma its best to have a smaller DB,
it will load much much faster.
